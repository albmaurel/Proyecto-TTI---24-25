Computes the velocity vector at the second position using the Herrick-Gibbs method.

This function implements the Herrick-Gibbs method to estimate the velocity vector at the second position vector (r2) given three position vectors (r1, r2, r3) and their corresponding Modified Julian Dates (Mjd1, Mjd2, Mjd3). It also checks for coplanarity and angular constraints, returning diagnostic information.

Parameters
r1First position vector (Matrix).
r2Second position vector (Matrix).
r3Third position vector (Matrix).
Mjd1Modified Julian Date corresponding to r1.
Mjd2Modified Julian Date corresponding to r2.
Mjd3Modified Julian Date corresponding to r3.
Returns
A tuple containing:
  • Reference to the computed velocity vector at r2 (Matrix&),
  • Angle between r1 and r2 (double),
  • Angle between r2 and r3 (double),
  • Coplanarity angle (double),
  • Error message string (string).
