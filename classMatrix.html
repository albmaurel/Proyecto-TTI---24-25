Matrix & Matrix::operator/ ( Matrix & matrix2)

Overloads the division operator to perform matrix division.

This operator divides the current matrix by another matrix by multiplying the current matrix with the inverse of the second matrix. It checks if the dimensions of the matrices are compatible for multiplication.

Parameters
matrix2The matrix to divide the current matrix by.
Returns
A reference to the resulting matrix after division.
Note
The function exits the program if the dimensions of the matrices are incompatible.
